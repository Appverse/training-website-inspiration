Install NPM modules and depencencies

This guide walks you through the process installing all needed node modules to start with Appverse HTML5 generator

Note: Nodejs and NPM package manager must be installed. See the Setup Tutorial

Install the Appverse HTML5 toolset

The easiest way to kickstart a new project with Appverse HTML5 is to scaffold a boilerplate app using Yeoman. Yeoman uses scripts called 'generators' to scalffold applications. You just need Checkout the Appverse HTML5 generator repository, and execute it with Yeoman. Just follow these steps:


Install Yeoman , Bower and Grunt

npm install --global yo bower grunt-cli

To make sure you have correctly installed Bower , Yeoman and Grunt

yo --version && bower --version && grunt --version

Get the appverse-html5 generator:

npm install -g generator-appverse-html5

Run appverse-html5 generator:

mkdir myapp
cd myapp
yo appverse-html5

Run your app

grunt server:open
